Install CHAT DDT di iPhone

Gunakan Safari untuk menambahkan aplikasi ke Home Screen.

CHAT DDT sudah dibuka sebagai aplikasi. Tidak perlu memasang ulang.
1
Buka halaman ini menggunakan Safari Menu Add to Home Screen tersedia melalui Safari iPhone.
2
Tekan tombol Bagikan Gunakan ikon kotak dengan panah ke atas pada toolbar Safari.
3
Pilih “Tambahkan ke Layar Utama” Atau “Add to Home Screen” sesuai bahasa iPhone.
4
Tekan Tambah Ikon CHAT DDT akan muncul di Home Screen dan terbuka tanpa bar Safari.
Setelah pertama kali dibuka dari ikon Home Screen, login satu kali di versi aplikasi karena penyimpanan sesi web app dapat terpisah dari tab Safari.
